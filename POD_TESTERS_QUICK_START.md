# Pod Testers Quick Start (Web)

This guide is for friends testing Gatherer in the browser.

## 1) Open the website URL

Ask the organizer for the latest Gatherer web link.

## 2) Sign in

1. Open `/auth`.
2. Enter your email and request a magic link.
3. Click the link from your email.
4. If callback fails, paste the full link URL into the fallback field on the auth page.

## 3) Join your pod

1. Open `Invites`.
2. Accept pending invite(s).
3. Open your pod and event details.

## 4) Report issues

Send screenshots, exact error text, browser name/version, and timestamp to the organizer.
